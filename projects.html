Instagram Staistics Manager

April 2020
As an Instagram influencer, I'm interested in how I can most effectively grow my account and the number of people I reach. This Python application connects with the Instagram API to generate useful statistics for Instagram influencers. It features a Tkinter user interface and dynamic Matplotlib graphs which help users to make well informed decisions when posting.
