Instagram Web Scraper
My Instagram account (@co14ers) is driven by user submissions which I then curate and repost. When reposting photos, I used to screenshot the original photo on Instagram and crop it on my phone. However, this method is time consuming, leaves room for cropping errors, and potentially lowers the image quality. The better alternative for reposting looks like retrieving the original image from the internet. When the Instagram post is viewed in a web browser, it prevents the user from copying or saving the image directly. However, there is a quick workaround by locating the url of the image in the HTML code and saving it from there. This process, however, though more accurate and high quality, tends to be even more time consuming. I believe that this second method can be automated. My idea is that this web scraper application would automatically download the highest quality version of the photo from a given url of an Instagram post. This would save me a lot of time reposting photos in the future.
